# NextEditor

Workspace for the `next-editor` package and a demo Next.js application.

The package is the product. The demo exists to exercise the published package
API in a real Next.js app.

## Structure

- `packages/next-editor`: installable package
- `apps/demo`: consumer app that uses the package locally
- `reference`: build spec and notes

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```
