# next-editor

Minimal inline editing layer for custom Next.js sites.

## Public API

- `definePage`
- `text`, `textarea`, `image`, `select`, `toggle`
- `EditorProvider`
- `EditorSidebar`
- `FloatingAdminBar`
- `EditableText`
- `EditableImage`
- `EditableRegion`
- `useEditor`

The host app owns routes, auth, persistence, and page-level content loading.
